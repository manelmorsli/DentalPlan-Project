# DentalPlan-Project
Site web vitrine et fonctionnel pour le cabinet dentaire DENTAL PLAN (Dr. Guezoui). Développé en HTML5, CSS3 et   JavaScript, il présente 8 traitements dentaires, intègre un formulaire de prise de rendez-vous et un espace   connexion/inscription.
